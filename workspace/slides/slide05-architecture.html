System Architecture

  • Mobile App (React Native + Expo) connects via WebSocket to Backend API (Express + SQLite)
  • BLE Mesh enables peer-to-peer device communication offline
  • Shared CRDT Library (LWW-Map, Vector Clock) syncs state across all nodes
  • Chaos Server (Flask) simulates flood events on network edges
