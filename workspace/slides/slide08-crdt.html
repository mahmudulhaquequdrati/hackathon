KEY FEATURE

Offline-First CRDT Sync

  • LWW-Map: Each supply record is a map of Last-Writer-Wins registers
  • Vector Clocks track causal ordering across all devices
  • Delta-Sync: Only changed records transmitted (under 10 KB)
  • Zero Central Arbiter — backend is a peer, not authority
  • Conflicts: Higher timestamp wins, tie-break on node ID
