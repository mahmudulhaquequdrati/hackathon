STACK

Technology Stack

  • Backend: Express.js + better-sqlite3 + WebSocket
  • Mobile: React Native 0.81 + Expo 54 + TypeScript
  • Database: SQLite on server and device (14 tables)
  • Security: Ed25519, x25519, JWT, TOTP (RFC 6238)
  • ML: Pure JS logistic regression (98.3% accuracy)
  • Sync: Custom CRDT (LWW-Map, Vector Clocks)
  • Mesh: BLE with store-and-forward relay
